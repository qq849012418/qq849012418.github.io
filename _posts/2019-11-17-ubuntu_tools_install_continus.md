## 1.多媒体处理类需求

### 录屏

#### SimpleScreenRecorder

屏幕上右键打开终端

添加源：

```Shell
sudo add-apt-repository ppa:maarten-baert/simplescreenrecorder
```

更新源：
```Shell
sudo apt-get update
```
安装：
```Shell
sudo apt-get install simplescreenrecorder
```
